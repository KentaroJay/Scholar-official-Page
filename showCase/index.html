文字定義について

文字は以下のサイズ、色で定義されています。それ以外の文字サイズ、色は使用しないようにしましょう。

文字サイズは、タイトル24px、本文 18px、行間 1.5pxで基本的には定義してください。

  • scssに記述する場合(@includeを用いる事によって、以下の形式を使用することができます)

    ※scssファイルに記述する場合、エディタの変換機能を用いて、それぞれのcssファイルに変換する必要があります。

    一通りクラス定義はcssファイルに書いてあるのでそれを利用してください

    1、セクションタイトル

    サンプル

    scss: @include text-section-title

     css: 
                                                
.text-section-title{
  padding-bottom: 15px;
  font-weight: bold;
  font-size: 24px;
  border-bottom: #000000 solid 1.5px;
  font-family: YuGothic Arial, sans-serif;
}

    この文字はセクションごとのタイトル（下線入り）で使用します

    2、本文

    サンプル

    scss: @include text-body

     css: 
                                                
.text-body{
  font-size: 18px;
  font-family: YuGothic Arial, sans-serif;
}

    この文字は本文で使用します。

    3、ヘッダーメニュー

    サンプル

    scss: @include text-menu

     css: 
                                                
.text-menu {
  font-size: 18px;
  font-family: YuGothic Arial, sans-serif;
  color: #868282;
}

    この文字はヘッダーメニューで使用します。

    4、ヘッダーのサブメニュー

    サンプル

    scss: @include text-sub-menu

     css: 
                                                
.text-sub-menu {
  font-size: 14px;
  font-family: YuGothic Arial, sans-serif;
  color: #868282;
}

    この文字はヘッダーメニューで使用します。

タグについて

タグは各自で用途や意味を調べながら使いましょう。今後案件をこなしていく中で、必須の知識になってくるので少しずつ慣れていきましょう

こちらのサイト（MDN）ではそれぞれのタグの用途、意味について詳しく解説されているので、参考にすると良いでしょう。

スペース、コンテナについて

それぞれのコンテンツの間のスペースや、コンテナは決められています。

見出しと本文の間は 40px

本文の下から次の見出しまでは 80px

で統一してください

container.scssでコンテナ幅の統一を行っています。

scssが書ける方は、@include containerで導入することができます。

また、cssの場合、以下のクラスを適用することで簡単に空白を調節することができます。

                
.c-container {
  width: 100%;
  margin: 80px auto;
  padding: 0 40px;
}

ブレイクポイントについて

768px（タブレットサイズ）で定義しています。端末によって配置をカスタマイズする場合（レスポンシブ対応など）は以下を使用しましょう。

cssの場合

                
@media (max-width: 768px) {
}

Scssの場合

@include sp()で使用することができます。
